常用语法：

  • 统计总数：SELECT COUNT(*) FROM table_name;
  • 统计某列的唯一值数量：SELECT COUNT(DISTINCT column_name) FROM table_name;
  • 统计某列的平均值：SELECT AVG(column_name) FROM table_name;
  • 统计某列的最大值：SELECT MAX(column_name) FROM table_name;
  • 统计某列的最小值：SELECT MIN(column_name) FROM table_name;
  • 统计某列的总和：SELECT SUM(column_name) FROM table_name;
  • 分组统计：SELECT column_name, COUNT(*) FROM table_name GROUP BY column_name;
  • 条件查询：SELECT * FROM table_name WHERE condition;
  • 排序查询：SELECT * FROM table_name ORDER BY column_name ASC/DESC;
  • 分页查询：SELECT * FROM table_name LIMIT offset, count;
  • 连接查询：SELECT * FROM table1 JOIN table2 ON table1.column_name = table2.column_name;
  • 子查询：SELECT * FROM table_name WHERE column_name IN (SELECT column_name FROM another_table);
  • 聚合查询：SELECT column_name, COUNT(*) FROM table_name GROUP BY column_name;
  • 分组查询：SELECT column_name, COUNT(*) FROM table_name GROUP BY column_name;

常用函数：

  • 字符串函数：CONCAT(str1, str2, ...)、SUBSTRING(str, start, length)、LENGTH(str)、UPPER(str)、LOWER(str)、REPLACE(str, old_str, new_str)
  • 日期函数：CURDATE()、CURTIME()、NOW()、DATE_FORMAT(date, format)、DATE_ADD(date, INTERVAL expr unit)、DATEDIFF(date1, date2)
  • 数学函数：ABS(number)、ROUND(number, decimals)、CEIL(number)、FLOOR(number)、RAND()、POWER(base, exponent)
  • 聚合函数：COUNT(*)、SUM(column)、AVG(column)、MAX(column)、MIN(column)
  • 条件函数：IF(condition, value1, value2)、CASE WHEN condition THEN value1 ELSE value2 END
  • 窗口函数：ROW_NUMBER() OVER (PARTITION BY column ORDER BY column)、RANK() OVER (PARTITION BY column ORDER BY column)、DENSE_RANK() OVER (PARTITION BY column ORDER BY column)
