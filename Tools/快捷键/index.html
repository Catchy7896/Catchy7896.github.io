Windows 10常用快捷键

常用快捷键

  • 打开文件管理器：Win + E
  • 打开任务栏：Win + 1/2/3/4/5/6/7/8/9
  • 切换窗口：Alt + Tab

其他

  • 1、显示桌面、最小化桌面：Win+D
  • 2、锁定桌面：Win+L
  • 3、打开文件管理器：Win+ E
  • 4、Windows原图截图：Win + Shift + S
  • 5、打开操作中心：Win+A
  • 6、打开搜索：Win + S
  • 7、时间轴、查看任务视图：Win + Tab
  • 8、打开运行窗口：Win + R
  • 9、打开任务管理器：Ctrl + Shift + Esc
  • 10、任务切换：Alt + Tab
  • 11、窗口排版：Win + →/↑/←/↓
  • 12、针对固定在快速启动栏中的程序，按照数字排序打开相应程序[Win+ 数字键（1-9）]
  • 13、打开开始菜单：Ctrl + Esc
  • 14、查看文件属性：Alt + 鼠标双击文件
  • 15、关闭当前程序或窗口：Alt + F4
  • 16、放大镜：
    • Win + + 打开系统放大镜
    • Win + - 缩小放大镜
    • Win + Esc 关闭放大镜
  • 17、虚拟桌面：
    • Win + Ctrl + D 创建虚拟桌面
    • Win + Ctrl + F4 关闭当前虚拟桌面
    • Win + Ctrl + ← 向左切换虚拟桌面
    • Win + Ctrl + → 向右切换虚拟桌面
  • 18、粘贴板：
    • Win +V 跳出粘贴板历史，图片也支持。

谷歌浏览器快捷键

常用

  • 收藏：Ctrl + D
  • 收藏所有标签页：Ctrl + Shift + D

其他

  • 同个窗口新建标签：Ctrl + T
  • 新建窗口：Ctrl + N
  • 关闭当前标签页：Ctrl + W
  • 关闭当前窗口：Ctrl + Shift + W
  • 关闭所有窗口：Ctrl + Shift + Q
  • 打开历史记录：Ctrl + H
  • 打开下载：Ctrl + J
  • 打开书签：Ctrl + E
